# 🚒Flame Detection System

## Project Overview
This project aims to develop a comprehensive security solution that automatically detects fire hazards and triggers suppression systems. The core strength of the system lies in its seamless integration of reliable hardware with a modern software ecosystem, ensuring homeowners remain in full control of their safety from anywhere.

## Requirement
- Docker desktop: Observe the log of three differents containers
- MQTT explore: Observe the log when sensors's data are published
- TablePlus: Observe what the database have been recieved

## Running project
Turn on terminal and run the command:
```
docker compose build --no-cache
```
